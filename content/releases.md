All releases of the CamillaDSP engine, with links to documentation and downloads.
Source code is in the [CamillaDSP repository](https://github.com/HEnquist/camilladsp).

CamillaDSP uses a three-part version number: **major.minor.patch**.
A new **patch** release (e.g. 3.0.0 → 3.0.1) is a bugfix-only update with no
changed functionality — you can upgrade without changing anything.
A new **minor** release (e.g. 3.0 → 3.1) adds features while keeping
backwards compatibility with existing configurations.
A new **major** release (e.g. 3 → 4) may introduce breaking changes that
require updating your configuration.
